# Заметки

- `src/` содержит автономные реализации `v1-v5`.
- `seird/` содержит старый эталонный код SEIRD, реконструированный переключаемый код старой архитектуры и текущую ветку `new/opt`.
- `tests/` содержит обёртки для TestU01 / PractRand, включая более поздние эксперименты `merge16` для round-robin склейки потоков.
- `results/` содержит ключевые сводки, на которых основаны таблицы и обсуждение в работе.
